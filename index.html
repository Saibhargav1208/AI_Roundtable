⚙ Customizing the Roundtable

Edit config.json Only
You never need to touch index.html or app.js. Everything is controlled by config.json in your GitHub repo. Just edit it and push — the page auto-fetches it on every load.
FieldWhat it does
debate_roundsHow many rounds of debate before conclusion (default: 9)
conclusion_roundtrue/false — whether to add a final problem statement round
inter_turn_delay_msPause between turns in ms (increase to avoid rate limits)
agents[ ]Add, remove, or change AI models. Each needs id, name, model, color, personality, style
agents[].modelAny model string from console.groq.com/docs/models
suggested_topics[ ]The dropdown topics shown below the text input
